## FindMyFlipperGUI

**Author:** Arxhsz and contributors  
**Repository:** https://github.com/Arxhsz/FindMyFlipperGUI

The native macOS Bluetooth implementation is compatible with the Flipper alert
service documented by the GUI implementation. It uses the GUI's published read
and write characteristic UUIDs and alert command payload while retaining a native
CoreBluetooth transport. No Electron or Bleak runtime is included in the Mac app.
